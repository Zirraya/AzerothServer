# 🐉 Моя Solo RPG сборка AzerothCore 3.3.5a

Добро пожаловать! Это готовая скомпилированная сборка сервера для одиночной игры.

## 🚀 Как запустить сервер (Инструкция по шагам):

1. **Скачайте файлы сервера:** 
   Перейдите во вкладку **Releases** справа и скачайте архив `server_files.zip` (он весит мало, там лежат `.exe` и настройки). Распакуйте его в удобное место на диске (желательно без русских букв в пути, например `E:\WoW_Server`).

2. **Скачайте карты мира (ОБЯЗАТЕЛЬНО):** 
   Поскольку карты весят очень много, скачайте папку `Data` по ссылке ниже и положите её прямо внутрь распакованной папки сервера (туда же, где лежат `worldserver.exe` и `authserver.exe`) либо извлеките из клиента игры:
   * 🔗 [СКАЧАТЬ ПАПКУ DATA С ЯНДЕКС ДИСКА](Файлы пока временно не доуступны)

3. **Настройка конфигурации:**
   Зайдите в папку `configs` внутри сервера. Переименуйте файлы `authserver.conf.dist` и `worldserver.conf.dist`, просто удалив окончание `.dist` (чтобы они назывались `authserver.conf` и `worldserver.conf`).

4. **База данных:** 
   Установите MySQL Server и импортируйте чистые дампы баз данных из папки `azerothcore-wotlk/data/sql/` вашего репозитория.

5. **Изменение реалмлиста в игре:**
   В папке с вашей установленной игрой WoW 3.3.5a найдите файл `Data/ruRU/realmlist.wtf` (или `enUS`), откройте его через Блокнот, сотрите всё и напишите:
   ```text
   set realmlist 127.0.0.1
   ```

6. **Запуск и создание аккаунта:**
   * Запустите вашу базу данных MySQL.
   * Запустите `authserver.exe`, а затем `worldserver.exe`.
   * Когда окно `worldserver.exe` полностью прогрузится, введите в него команду для создания вашего игрового профиля:
     ```text
     account create ЛОГИН ПАРОЛЬ
     ```
   * Запускайте игру через `WoW.exe` (не через лаунчер) и заходите под этими данными!
