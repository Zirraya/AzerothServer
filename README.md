# 🐉 Solo RPG сборка AzerothCore 3.3.5a

Исходник сервера.
   
   Приятной игры UwU
   
   Примечание: переведенно не все, особенно beastmaster. Если способности охотнтика пропадают это номрально это баг самого модуля.
   
##	 Имеющиеся модули:
	
	mod-ah-bot-plus - бот аукционист
	
	mod-ale - поддержка lua скриптов
	
	mod-aoe-loot - АОЕ лут
	
	mod-autobalance - баланс для количество игроков
	
	mod-individual-progression - постепенное открытие контента
	
	mod-npc-beastmaster - Повилитель Зверей который обучит любого персонажа призыву питомцу.
	
	mod-transmog - Трансмогрификатор рядом с тюрьмой Штормграда для Альянса. Для Орды пока не добавлен.

	## Изменения, которые были внесены в модули/сервер

* **Текущая фаза — 0** — сервер запущен в режиме постепенного открытия контента.
* **Сюжетный доступ к рейдам** — для открытия Логова Ониксии и Наксрамаса персонажам необходимо полностью завершить классические цепочки заданий.
* **Поэтапное открытие дополнений** — чтобы попасть в Запределье (The Burning Crusade), нужно пройти контент классической игры, а для путешествия в Нордскол (Wrath of the Lich King) — полностью освоить предыдущее дополнение.
* **Повелитель Зверей** — NPC-беастмастер частично переведен на русский язык и готов обучить призыву питомцев персонажа совершенно любого класса.
* **Соло-поиск подземелий (RDF)** — реализована возможность вставать в очередь в подземелья в одиночку. Система сразу переносит вас внутрь, позволяя зачищать подземелья в соло-режиме.

## Мой контент

* **Аедесра (Альянс)** и **Саротейн (Орда)** — NPC продают редких питомцев и ездовых животных, которых нельзя добыть внутриигровым путем (TCG, Blizzard Store, ивенты). Аедесра находится рядом с Дарнасом, Саротейн — рядом с Луносветом, где крылобеги. Продают пока Пламенеющего гиппогрифa.
* **Амулет Рыжего Саблезуба** — эпический предмет для слота шеи, который приходит на почту игроку по достижении 10 уровня вместе с письмом.
* **Белый Клык** — NPC из модуля beastmaster, находится рядом с Анклавом Кенария для Альянса, обучит игрока призыву питомцев.
* **Трансмогрификатор** — NPC из модуля transmog, находится рядом с тюрьмой Штормграда для Альянса.
		
## Контент, который будет реализован

* **Полеты в Старом Мире** — включение возможности летать на маунтах в локациях классического WoW.
* **Честь за убийство стражей** — начисление очков чести (Honor) при ликвидации NPC-охранников противоположной фракции.
* **Черный Клык** — кастомный NPC на основе модуля *beastmaster* для игроков Орды (аналог Белого Клыка).
* **Трансмогрификатор для Орды** — добавление NPC из модуля *transmog* в столицу Орды.
* **Обновление ассортимента** — расширение списка доступных редких маунтов и спутников у Аедесры и Саротейна.
