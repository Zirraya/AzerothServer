# 🐉 Solo RPG сборка AzerothCore 3.3.5a

Исходник сервера.
   
   Приятной игры UwU
   
   Примечание: переведенно не все, особенно beastmaster. Если способности охотнтика пропадают это номрально это баг самого модуля.
   
##	 Имеющиеся модули:
	
	mod-ah-bot-plus - бот аукционист
	
	mod-ale - поддержка lua скриптов
	
	mod-aoe-loot - АОЕ лут
	
	mod-autobalance - баланс для количество игроков
	
	mod-individual-progression - постепенное открытие контента
	
	mod-npc-beastmaster - Повилитель Зверей который обучит любого персонажа призыву питомцу. НПС - в Дарнасе рядом с Анклавом Кенария для Альянся. Для Орды пока не добавлен.
	
	mod-transmog - Трансмогрификатор рядом с тюрьмой Штормграда для Альянса. Для Орды пока не добавлен.
	
	mod-fly-in-old-world - Выключен. Не работает.
